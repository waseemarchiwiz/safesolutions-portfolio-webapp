## Introduction

SafeSolutionDemo is a React-based web application designed with an emphasis on modularity, performance, and scalability. It leverages the Vite build tool, Tailwind CSS for styling, and various utility libraries to enhance the development and user experience.

---

## Features

- State management with `react-hook-form` and `yup` validation
- Dynamic UI components using `Radix UI`, `Swiper`, and `Framer Motion`
- Interactive charts with `Recharts`
- Toast notifications with `React Toastify`
- Smooth animations powered by `Lottie`
- Lightweight, fast, and optimized for modern browsers

## Installation

Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/safesolutiondemo.git
   ```
2. Navigate to the project directory

   cd safesolutiondemo

3. Install dependencies:

   npm install

4. Start the development server

   npm run dev

## Avaliable Scripts

npm run dev
Starts the development server for local development.

npm run build
Builds the application for production.

npm run preview
Serves the production build locally for preview.

npm run lint
Runs ESLint for code quality checks.

npm run test
Runs tests using React Scripts (if any).
